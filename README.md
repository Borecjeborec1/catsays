# CATSAYS

# Installation

#### Install globally

    npm install -g catsays

# Usage

#### Global usage

    catsays <string> <options*> // catsays Hello World -black

    // catsays Hello World
    // *******************
    // * Hello World     *
    // *******************
    //   \
    //    \
    //     \
    //     |\__/,|   (`\
    //     |_ _  |.--.) )
    //     ( T   )     /
    //   (((^_(((/(((_/
